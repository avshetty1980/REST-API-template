## Setup

To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm run dev (runs both frontend and backend)
$ npm run start (runs server)
$npm run server (runs server in dev mode with nodemon)
$npm run client (only runs frontend)
```
